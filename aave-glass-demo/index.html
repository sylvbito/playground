▸ Minimal full-viewport displacement approach

1A canvas generates a low-resolution displacement map at 1/10th viewport size — e.g. 192×108 px for a 1920×1080 screen. Only ~20K pixels total. The lens shape — a rounded rect with sinusoidal gradient — is drawn at the current cursor position. Everywhere else is neutral (R=128, G=128, A=0) = no displacement.
2The SVG <filter> uses default bounds — covers the full element. feImage loads the tiny map scaled to (0, 0, vw, vh) with bilinear interpolation — SVG handles the upscaling for free.
3Single feDisplacementMap samples the map at each pixel. Neutral pixels → no displacement (content renders unchanged). Lens pixels → sinusoidal displacement → refraction. No chromatic aberration chains.
4feSpecularLighting reads the map's alpha channel as a bump map. A=255 inside lens → specular highlight. A=0 outside → transparent → no effect. Composited atop displaced content via feComposite arithmetic.
5Each frame: fresh filter id (Safari cache workaround). Map regenerated only when lens moves >3 px — Uint32Array.fill() for instant neutral reset.
4 filter primitives total: feImage → feDisplacementMap → feSpecularLighting → feComposite
1/10th viewport map: only ~676 lens pixels iterated per frame (vs 500K+ full-res)
Fresh filter id every frame — defeats Safari's filter output cache
No effect outside lens: neutral map pixels + transparent specular = unchanged content
<filter id="glass-42"
  filterUnits="userSpaceOnUse"
  colorInterpolationFilters="sRGB">
  <!-- Low-res map scaled to full viewport -->
  <feImage href="data:image/png;…"
    x="0" y="0"
    width="{vw}" height="{vh}"
    result="map"/>
  <feDisplacementMap
    in="SourceGraphic"
    in2="map"
    scale="45"
    result="displaced"/>
  <feSpecularLighting in="map"
    surfaceScale="3" specularConstant="0.6"
    lighting-color="#ffffff"
    result="spec">
    <fePointLight x="{cx}" y="{cy}" z="90"/>
  </feSpecularLighting>
  <feComposite
    in="spec" in2="displaced"
    operator="arithmetic"
    k2="1" k3="1"/>
</filter>
