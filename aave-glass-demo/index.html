▸ How the moving filter region works

1A canvas-generated PNG the size of the lens encodes displacement vectors — R=128 is neutral, edge values peak via sinusoidal gradient.
2The SVG <filter> uses filterUnits="userSpaceOnUse" so its x / y attributes place the lens in viewport coordinates.
3feImage loads the map at (0,0) relative to the filter region. feDisplacementMap displaces SourceGraphic using the map.
4Each frame: assign a fresh filter id, set x/y to the lerped cursor position — Safari's filter cache is defeated, lens moves.
Single <filter> applied to content container — browser composites only the lens region
Fresh filter id every frame — defeats Safari's filter output cache
Canvas-generated displacement map — lens‑sized PNG with sinusoidal gradient peaking at edges
feSpecularLighting + fePointLight for realistic glass highlight
<filter id="glass-42"
  filterUnits="userSpaceOnUse"
  x="{lensX}" y="{lensY}"
  width="{lensW}" height="{lensH}"
  colorInterpolationFilters="sRGB">
  <feImage href="map.png" result="map"/>
  <feDisplacementMap
    in="SourceGraphic"
    in2="map"
    scale="45"/>
  <feSpecularLighting in="map">
    <fePointLight x="..." y="..." z="90"/>
  </feSpecularLighting>
</filter>
