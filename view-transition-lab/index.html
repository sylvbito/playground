What This Is

Cross-document view transitions animate navigation between different HTML pages — no SPA, no JS router, no framework. Add one line of CSS (@view-transition { navigation: auto }) and the browser handles the rest. Customize with view-transition-name for shared-element morphing and ::view-transition-old/new for custom animations. Shipped in Chrome 126+, Safari 18+, Edge 126+.

Quick Start

/* Add to both pages — that's it */
@view-transition {
  navigation: auto;
}

/* Optional: name a shared element */
.hero-image {
  view-transition-name: hero;
}

/* Optional: customize the animation */
::view-transition-old(root) {
  animation: 0.3s ease both fade-out;
}
::view-transition-new(root) {
  animation: 0.3s ease both fade-in;
}
