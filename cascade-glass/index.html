Scroll-Triggered Reveal

Cards cascade into view as you scroll. The native animation-trigger CSS API (Chrome 145+) makes this declarative — no JavaScript needed.

CSS — native approach
.card {
  animation: reveal 0.8s ease both;
  animation-trigger: --scroll play-forwards;
  timeline-trigger-name: --scroll;
  timeline-trigger-source: view();
  timeline-trigger-range: entry 0% entry 100%;
}

@keyframes reveal {
  from {
    opacity: 0;
    transform: translateY(40px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
JS — progressive enhancement fallback
if (!CSS.supports('animation-trigger', '--x')) {
  const observer = new IntersectionObserver(
    (entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          entry.target.classList.add('revealed');
          observer.unobserve(entry.target);
        }
      });
    },
    { threshold: 0.15 }
  );
  document.querySelectorAll('[data-reveal]')
    .forEach((el) => observer.observe(el));
}

Perceptual Colour

Every gradient and accent on this page is authored in oklch() — a perceptually uniform colour space that ensures balanced lightness and consistent hue across the entire palette, avoiding the muddy transitions of sRGB.

OKLCH
04

Layered Depth

The glass effect is assembled from four layers: an ambient OKLCH gradient backdrop, morphing blur blobs for organic movement, a noise overlay for tactile texture, and the glass panels themselves with their inset highlights and gradient borders.
