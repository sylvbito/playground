01 · Select (base-select)

CSS
.form-select { appearance: base-select; &::picker(select) { background: var(--surface); border: 1px solid var(--border); border-radius: 10px; box-shadow: 0 12px 32px rgba(0,0,0,0.4); padding: 0.25rem; } option { padding: 0.5rem 0.75rem; border-radius: 6px; margin: 0.15rem; } option:hover { background: var(--accent-muted); } option:checked { background: var(--accent); color: white; font-weight: 500; } }
Usage Notes
appearance: base-select unlocks full CSS styling of <select> — the picker, options, and open/close state. No more div-wrapping hacks. Chrome 135+ and Edge 135+. Safari and Firefox are actively implementing. ::picker(select) targets the dropdown popup itself. :open pseudo-class fires when the picker is visible. option:checked styles the currently selected option. @starting-style enables open/close animations. Fallback: use @supports not (appearance: base-select) to provide the standard native select styling.
